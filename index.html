projects


2023

MerginMaps field survey environmet for commumities


What i learned

The aim of the project is to provide a software platform for the structured collection of infrastructure data from the municipality.

I decided to use a PostgreSQL Database for storing location data and further attributes.

For field survey i decided to use an open source plattform called MerginMaps based on python, redis, flask and nginx. There is a simple android App as well.

The complete software environmet is seperated in severel docker containers running in a virtual machine based on ubuntu linux.
